# LFIploit
A utility that bruteforces an LFI vulnerability using a standard or custom dictionary 

## Usage
- ### Windows\Unix
  <code>python lfiploit.py [-h] --url URL [--json] [--out OUT] [--dict DICT]</code>
- ### Unix
  <code>
    chmod +x lfiploit.py

    ./lfiploit.py [-h] --url URL [--json] [--out OUT] [--dict DICT]
 </code>
